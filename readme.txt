Displays the hooks.

== Description ==

= Search of hooks =
* Ascending order by first_call.
* Search by hook_name is possible.
* Search by called_by is possible.

== Installation ==

1. Upload `hooks-view` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
